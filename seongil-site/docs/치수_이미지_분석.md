## 2. 이미지별 상세 분석

### INLET_Size.png (인렛)
- **구성**: 3도 (등각도, 정면도, 측면도).
- **도면상 치수**:
  - 정면도: **606** (정면 외형 가로), **328** (정면 외형 세로).
  - 측면도: **216** (측면 깊이), **130** (측면 하단/전체 깊이), **33** (두께).
  - 기타: 573, 300, 555, 265 등 (개구부·상단 표기와 연계 가능).
- **재원표 연계**: Inlet_Summary의 정면 외형 가로(605), 정면 외형 세로(328), 개구부 가로(573), 개구부 세로(300), 상단 표기(555), 측면 깊이(216), 측면 하단 표기(130), 두께 표기(33)와 대응.
- **비고**: 도면 606 vs 재원표 605는 반올림/표기 차이일 수 있음. 개구부 가로/세로는 도면에 573, 300으로 표기 가능.

### SLB_A_Size.png (모터/사각형 팬)
- **구성**: 정면도 + 측면도, 치수 A·B·C·D·E 표기.
- **치수 의미**:
  - A: 프레임 가로(전체 폭).
  - B: 프레임 세로(전체 높이).
  - C: 본체 깊이(타공/장착면 관련).
  - D: 모터 돌출 깊이.
  - E: 전면 플랜지 두께.
- **재원표 연계**: SLB960_Dim은 「높이(mm), 지름(mm)」만 있어, 이 도면의 A~E와는 항목이 다름. **참고 도면**으로 두거나, 추후 재원표에 A~E가 추가되면 그때 매핑 권장.

### SLF_A_Size.png (환기팬 A타입)
- **구성**: 정면도 + 측면도, A·B·C·D·E 표기.
- **치수 의미**: SLF_A_Dimensions와 동일 — A/B 프레임 외곽, C 본체 깊이(타공), D 모터 돌출, E 전면 두께.
- **재원표 연계**: TYPE별 SLF-350A ~ SLF-960A 치수 카드와 함께 **치수 (A)** 섹션 상단에 배치하면 됨.

### SLF_D_Size.png (환기팬 D타입)
- **구성**: 원통형 팬 단면/등각도, A·B·C 표기.
- **치수 의미**: A=내경/날개 직경, B=외경, C=높이 — SLF_D_Dimensions의 「날개직경(mm), A, B, C」와 대응.
- **재원표 연계**: 형식이 D타입(SLF-200D 등)일 때 **치수 (D)** 섹션 상단에 배치.

### SLH_Size.png (후드)
- **구성**: 후드 등각도, A~F 치수 표기.
- **치수 의미**: A 높이, B 상단 길이, C 하단 폭, D/E 플랜지·리브, F 내부 개구(타공) 등 — Hood_Sizes의 A,B,C,D,E,F(타공)와 일치.
- **재원표 연계**: 후드 탭 TYPE 선택 시 스펙 카드 **위**에 이 이미지 배치.

### SLS_Size.png (셔터)
- **구성**: 정면도 + 측면도, A·B·T 표기.
- **치수 의미**: B=외곽 가로·세로, A=루버 개구, T=두께 — Shutter_Sizes 컬럼 순서 B, A, T와 일치.
- **재원표 연계**: 셔터 탭 TYPE 선택 시 스펙 카드 **위**에 배치.

## 4. 요약

- **즉시 연동 가능**: INLET_Size, SLF_A_Size, SLF_D_Size, SLH_Size, SLS_Size — 각 탭 치수/스펙 카드 **위**에 같은 이름으로 노출하면 됨.
- **SLB_A_Size**: 재원표에 A~E 치수 컬럼이 없으므로 참고 도면으로만 사용하거나, 추후 시트 보강 시 매핑하는 것을 권장.

위 배치대로 HTML/JS에서 이미지 경로만 연결하면 치수 도면이 제품소개 페이지에 반영됩니다.
