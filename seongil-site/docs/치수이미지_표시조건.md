# 치수 이미지 표시 조건 (방안 B: 좌 도면 / 우 카드)

Product Intro 섹션에서 치수 도면 이미지를 **언제 표시할지**에 대한 조건만 정리했습니다.

---

## 1. 공통 원칙

- **방안 B**: 해당 탭에서 **치수/스펙 카드가 실제로 그려지는 경우**에만, **좌측에 치수 이미지**를 표시합니다.
- 치수 데이터가 없거나 카드 영역이 비어 있으면 치수 이미지는 **표시하지 않습니다**.
- **이미지가 없을 때**: 이미지 공간을 할당하지 않고 **내용만 전체 폭**으로 나열합니다. 이미지가 있을 때만 좌(도면) / 우(내용) 2단 배치를 사용합니다.

---

## 2. 탭별 표시 조건

### 모터 (Motor)

| 항목 | 내용 |
|------|------|
| **치수 이미지** | **사용하지 않음** (모터 전용 치수 도면 이미지 없음) |
| **레이아웃** | 이미지 없이 **내용(스펙·치수 카드)만** 나열 |
| **치수 데이터** | SLB-960 → `SLB960_Dim`. SLB-350 등 → 동일 규격 SLF 치수 참조 시 `docs/모터_치수_환기팬_적용방안.md` 참고 |

---

### 환기팬 (Fan)

| 항목 | 내용 |
|------|------|
| **표시 조건 A** | 선택 형식에서 추출한 **baseType**이 `SLF_A_Dimensions`에 존재할 때 (치수 (A) 카드가 그려질 때) |
| **표시 조건 D** | baseType이 `SLF_D_Dimensions`에 존재할 때 (치수 (D) 카드가 그려질 때) |
| **이미지** | 조건 A 충족 시 `SLF_A_Size.png`, 조건 D 충족 시 `SLF_D_Size.png` |
| **동시 충족** | 한 형식이 A·D 둘 다 매칭되는 경우는 재원표 구조상 거의 없음. 있으면 A·D 각각에 대해 해당 이미지 1장씩 표시(세로 배치 권장) |
| **미표시** | 형식 미선택 또는 baseType이 A/D 어느 쪽에도 없을 때 → 치수 카드 없음 → 치수 이미지 미표시 |

- **baseType 추출**: 형식 문자열에서 정규식 `^(SLF-\d+(?:A|D))` 로 추출 (예: SLF-500A4-6 → SLF-500A, SLF-200D2-6 → SLF-200D).

---

### 후드 (Hood)

| 항목 | 내용 |
|------|------|
| **표시 조건** | **후드 탭**이 활성이고, TYPE이 선택되어 **스펙 카드**(A, B, C, D, E, F(타공))가 표시될 때 |
| **이미지** | `SLH_Size.png` |
| **미표시** | 후드 탭이 아닐 때(다른 탭 활성) |

- 후드 탭에서는 드롭다운에 항상 선택값이 있으므로, **후드 탭 활성 = 표시**로 처리하면 됩니다.

---

### 셔터 (Shutter)

| 항목 | 내용 |
|------|------|
| **표시 조건** | **셔터 탭**이 활성이고, TYPE이 선택되어 **스펙 카드**(B, A, T)가 표시될 때 |
| **이미지** | `SLS_Size.png` |
| **미표시** | 셔터 탭이 아닐 때 |

---

### 인렛 (Inlet)

| 항목 | 내용 |
|------|------|
| **표시 조건** | **인렛 탭**이 활성일 때 |
| **이미지** | `INLET_Size.png` |
| **미표시** | 인렛 탭이 아닐 때 |

- 인렛은 모델 선택이 없고 카드 리스트만 있으므로, **인렛 탭 활성 = 표시**로 처리합니다.

---

### 컨트롤러 (Controllers)

| 항목 | 내용 |
|------|------|
| **치수 이미지** | **없음** (재원표에 치수 시트 없음) |
| **표시** | 치수 도면 이미지 미사용 |

---

## 3. 요약 표

| 탭 | 표시 조건 | 이미지 |
|----|-----------|--------|
| 모터 | (이미지 없음) 내용만 나열 | — |
| 환기팬 | baseType이 SLF_A_Dimensions에 있음 → A도면 / SLF_D_Dimensions에 있음 → D도면 | SLF_A_Size.png / SLF_D_Size.png |
| 후드 | 후드 탭 활성 + TYPE 선택됨 | SLH_Size.png |
| 셔터 | 셔터 탭 활성 + TYPE 선택됨 | SLS_Size.png |
| 인렛 | 인렛 탭 활성 | INLET_Size.png |
| 컨트롤러 | (치수 이미지 없음) | — |

---

## 4. 레이아웃 (방안 B)

- **모터**: 치수 이미지 없음. 스펙/치수 카드만 나열
- **환기팬·후드·셔터**: **좌** 치수 도면 이미지, **우** 스펙/치수 카드 그리드
- **인렛**: 이미지가 가로로 길어 **상단(이미지) → 하단(내용)** 단일 열 배치
- **모바일**: 2단 해제 후 이미지 위 → 카드 아래 순서로 세로 배치

위 조건에 따라 JS에서 이미지 영역 표시/숨김 및 `img` src를 설정하면 됩니다.
