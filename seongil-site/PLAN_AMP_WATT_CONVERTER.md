# 자료실 페이지 – 암페어·와트 변환기 추가 계획

## 1. 목표
- 자료실(resources.html)에 **암페어(A) → 소비전력(kW)** 변환 도구를 추가한다.
- **AC 단상, 220V** 기준으로 **P(kW) = 220 × I(A) ÷ 1000** 을 사용한다.

---

## 2. 기능 정의

### 2.1 기준 및 공식
- **기준**: AC 단상, **전압 220V 고정** (입력 없음)
- **입력**: 전류 **암페어(A)**
- **출력**: 소비전력 **kW** (킬로와트)
- **공식**: `P(kW) = 220 × I(A) ÷ 1000`

### 2.2 UI
- 전압은 표시만: "AC 단상 220V 기준"
- **전류(A)** 입력란 1개
- **변환** 버튼 또는 입력 시 실시간 계산
- **결과**: "결과: xx.xx kW" 형식

### 2.3 유효성
- 숫자만 입력 허용, 0/음수 체크
- 빈 값이면 "전류(A)를 입력하세요" 등 안내

---

## 3. 페이지 구성

### 3.1 배치
- **자료 목록 위**에 새 섹션 추가 (상단이 더 눈에 잘 띔).
- 구성: **페이지 히어로** → **암페어·와트 변환기** → **자료 목록**.

### 3.2 HTML 구조 (resources.html)
- 페이지 히어로 아래에 **암페어·kW 변환기** 섹션 추가.
- 제목: "암페어·kW 변환기", 부제: "AC 단상 220V 기준으로 전류(A)를 소비전력(kW)으로 변환합니다."
- `#ampWattConverterContainer` 안에 app.js가 폼·결과를 주입.

- **공식**: AC 단상 220V 기준 `P(kW) = 220 × I(A) ÷ 1000`
